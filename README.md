Мини скрипт для создания эффекта затухания.
**Методы:**
```csharp
// Стандартное затухание (0.5с), пауза 2 секунды, стандартное появление (0.5с)
ScreenFader.Fade(2f);

// Затухание/появление по 1 секунде, пауза 3 секунды
ScreenFader.Fade(1f, 3f);

// Быстрый переход 0.5с затухание, без паузы, 0.5с появление
ScreenFader.Fade();

// Затухание, пауза, и вызов метода когда экран черный 
ScreenFader.Fade(1.5f, () => {
    Debug.Log("New Scene Load");
    SceneManager.LoadScene("Level2");
});
